This is a sketching tool for quickly generating profile/block drafts for weaving (and not a replacement for full drafting software).

It was built for sketching double-weave block so it currently only supports 2 blocks — A and B.

The custom fields support strings of A and B, or shorthand with letters and numbers to make longer stretches (e.g. A5 instead of AAAAA).

The weft custom fields also support C and D in the string to represent rows with both light or both dark on the top layer.
